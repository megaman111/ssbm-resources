ℹ️ External SLP-to-MP4 tools (Dolphin-based)

To convert Slippi replays to video files, use one of these tools:

  • BetterVODs — Desktop app that downloads Twitch VODs and gives them a YouTube-like feel for VOD review. Requires ffmpeg.
  • slp2mp4 — Python tool, converts .slp files to .mp4 using Dolphin. Supports batch conversion.
  • slp-to-video — Another Python converter using Dolphin playback.
  • Slippi2Video — Automates OBS recording of Slippi replays.

slp2mp4 is the most straightforward for .slp → video. BetterVODs is the move for Twitch VOD review.
